BOSS LEVEL

WPL2 Case

Mijn grootste uitdaging dit semester

01

Beschrijving van de Case

Tijdens WPL2 werkte ik aan de verdere uitbouw van mijn persoonlijk portfolio als professioneel online visitekaartje. De opdracht bestond uit het volledig herbouwen en uitbreiden van mijn bestaande WPL1-portfolio naar een portfolio dat klaar is voor mijn zoektocht naar een stageplaats in WPL3 & WPL4.

Het portfolio moest voldoen aan technische vereisten (HTML, CSS, JS), responsive zijn voor alle schermformaten, gepubliceerd worden via een publieke URL, en nieuwe secties bevatten: een WPL2 case-beschrijving, reflectie, en vrije sectie. Ik koos bewust voor een retro arcade/Tekken-thema om mij te onderscheiden van generieke portfoliosites.

02

Mijn Aandeel

Ik nam volledig de leiding over het ontwerp en de technische uitwerking. Ik begon met een moodboard en wireframes voor het nieuwe Tekken-thema, en werkte daarna stap voor stap de HTML-structuur, CSS-stijlen en JavaScript-interacties uit.

Concreet: ik ontwikkelde het pixel-font systeem, de health bars, de retro scanlines-overlay, het responsive hamburger menu, de projectkaarten met "ROUND"-stempel, en alle animaties. Alle code is eigenhandig geschreven in Visual Studio Code en gepusht via GitHub met regelmatige commits.

03

Wat ik heb geleerd

Dit project leerde mij om een sterk concept van begin tot einde door te trekken. Ik leerde hoe ik CSS-animaties en pseudo-elementen kan inzetten voor visuele effecten zonder zware bibliotheken. Responsive design werd concreter: grid-layouts, media queries en mobile-first denken.

Ik leerde ook dat documentatie (README, bronvermelding, privacy) even belangrijk is als de code zelf. En dat creatieve keuzes moeten onderbouwd worden — niet alleen mooi, maar ook functioneel en toegankelijk.
